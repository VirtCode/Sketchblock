Sketchblock

An alternative to the Windows Ink Sketchpad

Download Now

Sketchblock is a simple drawing application for Windows Ink. It's designed to be simple and lightweight but still functional and easy to use. Since the official Windows Ink Sketchpad was removed with the latest update, it is especcialy required to find a good replacement, so I made one. However it is not intended to use this programm for professial use or as an art programm because it has only basic functionality. Although it is good enough to make short notes at work or so. It is completly free to use and is available for download in the download section.
